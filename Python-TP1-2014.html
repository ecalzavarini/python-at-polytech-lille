Introduction aux commandes Linux¶

Avant de dbuter notre introduction au langage de programmation Python, nous devons familiariser avec l'environnement du systeme d'explotation Linux.

La premire tape est d'apprendre les commandes les plus courants. Ces commandes peuvent tre taps sur une console Linux.

Pour savoir comment accder une console sur la distribution Linux Debian dont vous disposez, voir: https://wiki.debian.org/fr/Console ou simplement recherchez l'application "terminal" sur le menu des applications.

La commande pwd¶

La commande "pwd" affiche le rpertoire courant, c'est a dire le rpertoire o l'on se trouve au moment o on tape la commande.

La commande ls¶

La commande "ls" permet de lister ce que contient un rpertoire. Il y a plusieurs manires de l'utiliser :

"ls" donne la liste brute de ce qui se trouve dans le rpertoire

"ls -l" donne la liste de ce qui se trouve dans le rpertoire avec des informations complmentaires (droits de lecture, criture et excution, propritaire, taille, date de cration ou de dernire modification).

"ls -a" liste tous (penser au mot "all") les fichiers du rpertoire, y compris les fichiers cachs. Cette option est trs utile lorsque l'on se trouve dans son rpertoire personnel car il contient les fichiers de configuration de l'utilisateur dont les noms commencent gnralement par un point et seule l'option -a permet de dtecter leur existence.

La commande mkdir¶

La commande "mkdir" permet de crer un rpertoire.

"mkdir nom_repertoire" cre le rpertoire ayant pour nom nom_repertoire.

La commande cd¶

La commande "cd" permet de se dplacer dans l'arborescence de repertoires. Il y a plusieurs manires de l'utiliser :

"cd nom_repertoire" permet d'aller dans le rpertoire nom_repertoire

"cd .." permet de retourner dans le rpertoire parent (i.e. Celui qui est au-dessus dans l'arborescence de rpertoires).

L'éditeur de texte emacs¶

Pour crer ou modifier un fichier "emacs -nw nom_fichier.txt"

Pour sauvegarder tapez la combinaison de touches suivante: Ctrl-x Ctrl-s.

Pour quitter emacs: Ctrl-x Ctrl-c.

La commande rm¶

La commande "rm" permet de supprimer un fichier :

"rm nom_fichier" supprime le fichier nom_fichier si il se trouve dans le rpertoire courant.

"rm r" supprime un rpertoire et ses sous rpertoires.

La commande top¶

La commande "top" affiche en continu des informations dcrivant l'activit du systme. Elle permet surtout de suivre les ressources que les processus utilisent (quantit de mmoire, pourcentage de CPU...). Sous top il est possible d'expdier de manire interactive un signal un processus, par exemple afin de le stopper, en tapant "k". top demande ensuite quel signal il doit envoyer et 15 (SIGTERM) est le signal par dfaut qui met fin un processus, 9 (SIGKILL) est beaucoup plus brutal et viter. Pour quitter top, appuyer simplement sur la touche "q".

Archivage de données (tar)¶

La commande "tar" gre des archives, contenant chacune au moins un rpertoire ou fichier.

Vous aurez souvent besoin de "tar xzf nom_du_fichier.tar.gz" , qui dcompacte une archive au format .tar.gz ou .tgz. L'extension .tar.gz ou .tgz indique que le fichier est une archive tar et qu'il est compact. Les arguments employs dans la commande prcdente ( xzf ) peuvent tre ainsi compris:

x (extraction) dclenche l'extraction de certains fichiers d'une archive (lorsque l'on ne spcifie pas les noms des fichiers que l'on souhaite extraire de l'archive, tar les extrait tous)

z dcompacte l'archive grce la commande gzip

f traite un fichier-archive dont le nom suit (ici: nom_du_fichier.tar.gz )

Si je me trouve dans le rpertoire /home/delcros/ la commande suivante crera une archive du rpertoire /home/delcros/personnel :

"tar cvzf personnel.tgz personnel"

N = len(M)

#Programme 
for i in range(0,N-1):
    for j in range(i+1,N):
